# Getting started
Running solution with shell script

## Start Zookepper and Kafka

```bash
make docker-compose-up
```

## Create Topics

```bash
make topic-creates
```

## List Topics

```bash
make topic-list
```

## Describe Topics

```bash
make topic-describe
```
